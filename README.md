# Active Thermography for Non-Destructive Determination of Bud Mortality for Grapes

This repository contains the complete research codebase and data organization for a two-part study on **non-destructive grape-bud mortality assessment** using an **engineered active thermography system** and **machine learning models**.

The research is presented as **two independent but connected manuscripts**, each with its own scope, methods, and reproducible pipelines.
